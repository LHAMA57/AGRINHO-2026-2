📚 Flashcards de Estudo

Aprenda de forma rápida e organizada

HTML

O que é HTML?

HTML é uma linguagem usada para criar a estrutura das páginas web.

CSS

Para que serve?

CSS estiliza páginas, criando cores, layouts e animações.

JavaScript

Qual sua função?

Adiciona interatividade e comportamentos ao site.

GitHub

O que é?

Plataforma para guardar, compartilhar e controlar códigos.

DOM

O que é?

O DOM permite que o JavaScript altere elementos do HTML da página.

Tag HTML

O que representa?

São comandos usados para criar elementos como textos, imagens e botões.

Classe CSS

Para que serve?

Organiza estilos que podem ser aplicados em vários elementos.

Arquivo .JS

Qual função?

Guarda códigos JavaScript que controlam ações e interações.

Arquivo CSS

Por que usar?

Separa o design da programação, deixando o projeto organizado.

Index.html

O que é?

É normalmente a página inicial de um site.

Git

Qual sua função?

Controla versões do código e registra alterações do projeto.

Hospedagem

O que faz?

Coloca o site disponível para outras pessoas acessarem.

Viewport

Para que serve?

Ajuda o site a se adaptar ao tamanho da tela.

UX

O que significa?

Experiência do usuário: tornar o site fácil e agradável de usar.
